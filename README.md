Python libraries and scripts for converting numbers using the [Mnemonic Major System].

INSTALL
-------

    pip install major_system


USAGE
-----

    major_words --help


RELEASE
-------

Released by gitlab CI whenever a commit to master is made at [the gitlab repository].
